# Lun UI/UX 기획서 v1

## 현재 문제 진단

1. **재미가 없다** — 평범한 채팅창 UI
2. **에이전트들이 정적이다** — 응답하고 끝, 서로 반응 없음
3. **시각적 차별화 약함** — 색상 외에 개성 없음
4. **메타 정보 부족** — 누가 빠른지, 어디 강한지 안 보임
5. **단조로운 레이아웃** — 사이드바 + 채팅이 전부

## 핵심 메타포

**"AI 에이전트들의 라운드 테이블 토론"**

- 사용자: 의장
- PM 에이전트: 진행자 (moderator)
- 다른 에이전트들: 패널리스트
- Lun은 그 회의장

## 디자인 원칙

| 원칙 | 의미 |
|------|------|
| **에이전트 = 캐릭터** | 각자 아바타, 색상, 스타일이 보이게 |
| **공간감** | 평면 채팅 대신 회의실/스테이지 느낌 |
| **실시간성** | 정적 텍스트 → 타이핑/스트리밍/반응 |
| **메타 가시성** | 응답 시간, 토큰, 비용이 자연스럽게 |
| **상호작용** | 사용자가 토론에 끼어들 수 있게 |

## 3가지 모드

### 1. Round Table (Discuss)
- PM이 중앙 상단
- 패널 카드 좌우 배치
- 라운드별로 무대처럼 펼쳐짐
- 마지막에 종합 카드

### 2. Stream (Chat)
- 카톡 그룹채팅 스타일
- PM이 주된 대화 상대
- 다른 에이전트는 PM이 호출할 때 등장

### 3. Compare (Ask)
- 컬럼별 나란히 비교
- 공통점 하이라이트
- 차이점 강조
- 투표 결과 바

## 화면 우선순위

1. **Round Table View** (Discuss) — 핵심 차별점
2. **Chat Mode** (PM-led) — 일상 사용
3. **Compare View** (Ask) — 비교 분석
4. **Onboarding** — 첫 인상
5. **Mobile** — 반응형
